# Weather Widget for Frogcast
Quos Group's weather widget using data from the Frogcast API for its forecasts.

## Overview
A widget displaying the weather for the 3 to 5 days to come.
This widget is meant to be free to use for anyone wanting to integrate it into their website.

## Installation
### Adding the widget to a project :
To add the widget to a web page its html needs to hold 2 components : a script element referencing the hosted WeatherWidget.js and a weather-widget element with the chosen settings. Removing the branding slot doesn't affect the widget's performance but shouldn't be done as it is part of the branding.
